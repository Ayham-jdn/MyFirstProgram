# Simple Rock-Paper-Scissors Game

This is a **C# console game** implemented using **Object-Oriented Programming (OOP)** principles.

## Features

- Play against the computer
- Input validation
- Game logic encapsulated in `Game` and `Player` classes
- Easy to extend for scoring or multiple rounds

## Classes

- `Player` – Represents a human or computer player.
- `Game` – Handles the game logic and determines the winner.
- `Program` – Entry point for the console application.

## How to Play

1. Run the application.
2. Enter your name.
3. Choose ROCK, PAPER, or SCISSORS.
4. See the results against the computer.
5. Optionally play multiple rounds.
